## 🎓 **KEY CONCEPTS TO UNDERSTAND**

### **1. Frontend (What User Sees)**
- React: UI framework
- Next.js: React framework
- TypeScript: Type-safe JavaScript
- Tailwind CSS: Styling

### **2. Backend (What Powers It)**
- Django: Python web framework
- REST API: How frontend talks to backend
- SQLite: Database for storage (Default) -> MySQL: Production Database
- JWT: Authentication method

### **3. How They Talk**
- Frontend sends: "Give me all products"
- Backend: Gets from database, sends JSON
- Frontend: Displays on screen

### **4. Data Flow**
```
User clicks → Frontend reacts → Sends request to API
→ Backend processes → Database stores → API responds
→ Frontend updates → User sees result
```
