## **TROUBLESHOOTING CHECKLIST**

### **If Something Goes Wrong:**

```
□ Check System/App
  - Is it running?
  - Any error messages?
  - Is internet working?

□ Check Data
  - Is data saved?
  - Can you see invoices in Reports?
  - Is calculation correct?

□ Check User
  - Is correct person logged in?
  - Do they have permission?
  - Are they using correctly?

□ Check Process
  - Are steps correct?
  - Was data entered right?
  - Is everything complete?

□ Restart
  - Close app
  - Reopen
  - Try again
  - Often fixes issues!

□ Contact Support
  - If still not working
  - Keep error details
  - Describe what happened
```
